# CLAUDE.md — Claude-skills-repo

Published collection of Claude Code skills (l3ad3r1/Claude-skills-repo). Markdown content — no build, no tests.

## Rules
- Every new or updated skill MUST: (1) include its own documentation inside the skill folder, (2) credit the upstream source it derives from, (3) be pushed to this repo AND shipped as a GitHub release (`gh release create` — mark latest).
- Keep skill frontmatter valid (`name:`, `description:`); the description is the trigger — write it for matching, not marketing.
- Never paste secrets or API keys into skill files; they are public on release.
- Prefer editing an existing skill over forking a near-duplicate; check the folder list first.
