This skill documents and adapts the open-source "Agent Reach" CLI.

Upstream project: https://github.com/Panniantong/agent-reach
Upstream license: MIT License
Author: Panniantong (pnt01@foxmail.com, X: @Neo_Reidlab)

The SKILL.md instructions describe how to install and operate the upstream
Agent Reach CLI and its bundled backends (Jina Reader, twitter-cli, yt-dlp,
OpenCLI, rdt-cli, bili-cli, gh, feedparser, Exa via MCP). The CLI itself is
installed from upstream at runtime (pipx / pip from the GitHub archive) and is
NOT vendored in this repository. This package only provides the skill
instructions in the Agent Skills / Hermes SKILL.md format and does not modify
the upstream project.
