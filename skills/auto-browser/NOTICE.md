This skill documents and wraps the open-source "auto-browser" project.

Upstream project: https://github.com/LvcidPsyche/auto-browser
Upstream license: MIT License
Author: LvcidPsyche

The auto-browser source is NOT vendored here. Run the stack from the upstream
repo (`docker compose up`) and install the MCP stdio bridge from PyPI
(`uvx auto-browser-mcp`). This skill only provides setup guidance and
troubleshooting for the Agent Skills / Hermes SKILL.md format and does not
modify the upstream tool.
